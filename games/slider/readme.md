# Slider

## One of those slide puzzles

-   Random Images!
-   Starts with a random piece missing!
-   First puzzle is 2 x 2, and the dimensions increase by one each time you complete a puzzle!
-   Goes on forever!
-   No points to score!
-   Not even a timer!
-   Try to stay sane!
